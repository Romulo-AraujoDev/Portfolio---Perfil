Sobre mim

Sou um desenvolvedor de software com foco em soluções web modernas e escaláveis. Desde o início da minha jornada na tecnologia, desenvolvi um forte interesse por criar aplicações que unem desempenho, usabilidade e um bom design. Ao longo do tempo, mergulhei em diversas tecnologias do ecossistema JavaScript, tanto no frontend quanto no backend, sempre buscando evoluir e entender o impacto real do código no dia a dia dos usuários.

Formação

Tecnólogo em Sistemas para Internet Centro Universitário de João Pessoa (UNIPÊ) Período: [2023] – [2025] Durante o curso, adquiri conhecimentos sólidos em desenvolvimento web, arquitetura de software, usabilidade, segurança da informação e engenharia de software. O curso me deu uma base acadêmica consistente, complementada com práticas de programação orientada a objetos, modelagem de banco de dados e desenvolvimento de aplicações web responsivas.

Cursos

Programação Web Curso com foco em desenvolvimento frontend e backend para aplicações web modernas. Estudei HTML, CSS e JavaScript com foco em boas práticas, semântica e responsividade, além de versionamento com Git e integração com APIs REST. Desenvolvimento Full Stack Treinamento completo com foco no ecossistema JavaScript. Aprofundei meus conhecimentos em React e TypeScript no frontend, além de Node.js, Express e MongoDB/MySQL no backend. Também trabalhei com autenticação, JWT, deploy com serviços como Vercel e Heroku, e versionamento com Git.
